PókeApp

Angular Ionic TypeScript Sass

Projeto simples que consome a API do Pokémon, desenvolvido para aprimorar minhas habilidades em Angular e Ionic.
O código está organizado em componentes por função (lista, detalhes, favoritos e cards), com um serviço centralizado para chamadas HTTP, usando injeção de dependência para facilitar testes e reutilização.
A interface é limpa, simples e responsiva, focada na melhor experiência do usuário. Priorizei código claro e modular, aplicando padrões como Service Pattern, Dependency Injection, Observer Pattern com RxJS e Separation of Concerns para manter a lógica desacoplada da apresentação.
Implementei testes unitários com mocks, que garantem qualidade sem depender da API real.

App de receitas

React Javascript

Este foi o último projeto do módulo de front-end na Trybe, com uma proposta diferente dos anteriores: o desenvolvimento voltado para mobile. Foi, até aquele momento, o projeto mais completo que desenvolvi, aplicando na prática tudo que aprendi nos seis primeiros meses de contato com programação.
O desenvolvimento foi em grupo, junto com Douglas Ferreira e Daniel de Sousa Rodrigues. Optamos por utilizar Context API para gestão de estado e comunicação entre componentes. Trabalhamos com metodologias ágeis, cumprindo e até antecipando nossas metas — finalizamos o projeto dois dias antes do prazo.
O layout foi todo construído com CSS puro, com exceção do carousel na página de detalhes, onde utilizamos uma biblioteca para facilitar. Por ser um projeto desenvolvido durante minha fase inicial nos estudos, a interface não está tão refinada, mas o foco foi consolidar os conhecimentos em lógica, React e boas práticas de desenvolvimento mobile-first.
Foi uma experiência extremamente enriquecedora, que marcou minha evolução na transição para o desenvolvimento back-end.

PókeApp

React Javascript

Projeto desenvolvido durante minha fase de estudos na Trybe, como resposta a um desafio proposto em sala pelo instrutor Ícaro Harry para a turma 08. A ideia era criar um contador regressivo que marcasse o tempo dos intervalos durante as aulas.
Aproveitei a proposta como uma oportunidade de acelerar meu aprendizado em React, que na época eu havia começado a estudar há menos de uma semana. Foi um fim de semana intenso, de muitos desafios, onde pude consolidar conceitos fundamentais do desenvolvimento com React, como estado, ciclo de vida dos componentes e manipulação de tempo.
Como funcionalidade extra, incluí também uma contagem regressiva até a data da formatura da turma 08, a "turma do infinito".
Apesar de simples visualmente, foi um projeto muito significativo na minha jornada, especialmente por representar uma das minhas primeiras experiências práticas com React.
